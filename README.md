# Smart Assignment Manager

## 🚀 Project Overview
The **Smart Assignment Manager** is a **Python-based command-line application** that helps users track and manage their assignments or tasks efficiently.  
It allows users to **add, view, prioritize, complete, and delete assignments**, all with a simple menu-driven interface.

This project is ideal for students, professionals, or anyone who needs to manage deadlines effectively.

---

## 🛠 Features

- **Add Assignments** – Save tasks with a title and deadline.  
- **View Assignments** – See all assignments with **priority status**.  
- **Automatic Priority System** – Assignments are labeled as **High, Medium, Low, or Overdue** depending on the deadline.  
- **Mark as Complete** – Update assignments to indicate completion.  
- **Delete Assignments** – Remove assignments that are no longer needed.  
- **Persistent Storage** – Tasks are stored in a SQLite database (`assignments.db`) so your data is saved between sessions.  

---

## 📌 Technology Stack

- **Programming Language**: Python 3.x  
- **Database**: SQLite (built-in Python library)  
- **Libraries**: Only standard Python libraries (no external dependencies)  

---

## 🎯 Problem Statement

Many students or professionals miss deadlines or forget assignments because tasks are scattered or not tracked.  
This tool centralizes task management, helping users **stay organized and meet deadlines**.
